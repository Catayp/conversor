Proyecto conversor de unidades

Este proyecto se realizo para facilitar la conversion de unidades. De esta forma, las conversiones que se manejaran seran: longitud, masa, tiempo. Se desarrollo el proyecto con simulaciones y animaciones para facilitar el aprendizaje y tener mas dinamismo.
